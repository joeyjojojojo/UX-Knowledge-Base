4. 基线网格的设计原理

为什么选择24px作为基线？

数学考虑

24px可以被2、3、4、6、8、12整除，便于各种字号对齐。

阅读体验

24px提供了舒适的行间距，特别适合16px的正文字体。

设计灵活性

可以使用12px（半基线）进行更精细的调整。

常见基线值

• 20px - 适合小尺寸设计
• 24px - 最常用的标准
• 28px - 适合大字号内容
• 32px - 适合展示型设计

计算公式

理想基线 = 正文字号 × 1.4-1.6
例：16px × 1.5 = 24px

🎯 实际应用建议

在实际项目中，建议先确定基线高度，然后让所有文字元素的line-height都是基线的整数倍。对于不同字号，可能需要添加少量margin-top来微调基线对齐。
